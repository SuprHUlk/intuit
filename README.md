# Intuit Assignments

This repository contains two programming assignments demonstrating different design patterns and programming paradigms.

## Assignments

### [Assignment 1: Producer-Consumer Pattern](./assignment1/README.md)

Java implementation of the producer-consumer pattern with thread synchronization.

**Tech Stack:** Java 11, Maven, JUnit 5

### [Assignment 2: Sales Data Analysis](./assignment2/README.md)

Python application for analyzing sales data using functional programming paradigms.

**Tech Stack:** Python 3.8+, Pandas, Pytest

## Quick Start

Navigate to each assignment folder and follow the README instructions for setup and execution.
